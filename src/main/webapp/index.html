Bienvenue sur Agora

Agora est une application web pour gérer tous vos clients et constats. Que vous soyez chez votre client ou au bureau, sur smartphone ou sur votre ordinateur Agora sera présent.

Etape 1 - Création ou sélection de votre client

Lorsque vous créez un constat, vous devez avant tout sélectionner un client. Si celui-ci n'apparaît pas dans votre liste client, alors vous devez créer sa fiche client.

Etape 2 - Renseignez les informations du constat

A partir de votre smartphone, vous pouvez ajouter des fichiers audios, images et autres. Seul votre smartphone suffit, fini le bloc note ou le dictaphone.

Etape 3 - Sauvegardez votre constat

N'ayez plus peur de perdre vos données. Une fois votre constat sauvegardé, il est accessible à partir de n'importe quel endroit.
